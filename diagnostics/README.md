# Archivos de Diagnóstico

Esta carpeta contiene scripts de diagnóstico y debugging utilizados para identificar y resolver problemas en la extensión ArbitrageAR-USDT.

## Archivos

- `diagnostico_completo_config.js` - Diagnóstico completo de configuración
- `diagnostico_completo_problema.js` - Diagnóstico del problema identificado
- `diagnostico_dolar_config.js` - Diagnóstico específico de configuración del dólar
- `diagnostico_popup_background.js` - Diagnóstico de comunicación popup-background

## Uso

Estos archivos se utilizan para debugging y no son parte del código de producción. Contienen logging detallado y pruebas específicas para identificar problemas.

## Limpieza

Después de resolver los problemas, estos archivos pueden ser eliminados o movidos a un repositorio de archivos históricos.
